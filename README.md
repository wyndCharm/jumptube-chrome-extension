# JumpTube

JumpTube is a Chrome extension that helps you navigate YouTube videos using chapters and segments.

## Features
- Click to jump between chapters
- Floating panel on YouTube watch pages
- Keyboard shortcuts for faster navigation

## Privacy
JumpTube does not sell personal data.
The extension runs only on YouTube pages.
